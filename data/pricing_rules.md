# Northstar Pricing and Packaging Rules
 
Effective date: 2025-08-01
 
## Plans
Starter Plan:
- USD 2,000 per month
- Up to 3 data sources
- Up to 25 users
- Includes dashboards and CSV uploads
- Does not include forecasting
 
Growth Plan:
- USD 6,500 per month
- Up to 8 data sources
- Up to 100 users
- Includes dashboards, CSV uploads, customer segmentation, inventory alerts, and forecasting
 
Enterprise Plan:
- Starts at USD 15,000 per month
- Unlimited data sources
- Unlimited users
- Includes all modules
- Dedicated customer success manager
- Custom security review support
 
## Discounts
Annual contracts receive a 12% discount.
 
Nonprofit customers receive a 20% discount.
 
Discounts cannot be stacked. If multiple discounts apply, use the highest single discount.
 
## Implementation Fees
Starter implementation fee: USD 3,000 one-time.
Growth implementation fee: USD 8,000 one-time.
Enterprise implementation fee: USD 20,000 one-time.
